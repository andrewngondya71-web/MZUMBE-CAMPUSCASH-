# CampusCash Web# Campuscash

Campuscash is a secure digital financial platform and marketplace for university students.

## Features
- Student Marketplace
- Digital Wallet
- Savings Goals
- Budget Planner
- Financial Education
- Secure Authentication
- Transaction History
- Notifications

## Frontend
The frontend provides:
- User registration and login
- Marketplace interface
- Digital wallet
- Budget planner
- Savings goals
- User profile

## Backend
The backend manages:
- User authentication
- Database operations
- Product management
- Wallet transactions
- Payment integration
- API services
- Data security and encryption

## Technology Stack
- Frontend: Flutter, HTML, CSS, JavaScript
- Backend: Node.js, Firebase
- Database: Firebase Firestore

## Vision
To empower university students through secure digital finance, entrepreneurship, and financial literacy.
